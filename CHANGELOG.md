# Changelog

## v2.0.1.0-preview

稳定性与体验修补版本。

### 改进

- 改进 TrayHost 后台媒体监听与状态变更时的异常兜底。
- 改进切歌、播放状态变化、媒体会话变化时的刷新稳定性。
- 改进主磁贴与次级磁贴更新流程，单个磁贴更新失败不再影响整体刷新链路。
- 保留小磁贴 / 大磁贴轮播显示，同时为图片 URI 增加版本参数以降低 Shell 图片缓存影响。
- 日志写入改为 FullTrust 进程更稳定的 `System.IO` 写入方式。
- 改进状态快照读写，降低 UWP 读取半写入快照的概率。
- UWP 页面背景跟随系统主题，修复窗口拉伸时边缘可能透明的问题。

### 修复

- 修复部分 UWP WinRT 异步调用缺少 `using System;` 导致的编译诊断。
- 修复若干 nullable / 状态同步 / 资源释放相关细节。
- 修复部分项目文件与清单版本、缩进和引用配置问题。

### 说明

- 该版本仍为 preview。
- VS2022 调试 UWP + FullTrust 混合项目时，如遇调试期 `SEHException`，建议关闭 XAML 热重载后再调试。
- 最终运行表现建议以 MSIX Package 部署运行结果为准。

## v2.0.0.0-preview

- 首个 UWP + FullTrust TrayHost + MSIX Package 混合架构预览版本。
- 引入托盘常驻、后台媒体监听、主磁贴 / 次级磁贴更新、开机自启、状态快照与日志等能力。
